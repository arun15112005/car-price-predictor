# Car Price Predictor

A machine learning web application that predicts the resale value of used cars.

## Features
- Company selection
- Car model selection
- Fuel type selection
- Manufacturing year
- Kilometers driven

## Machine Learning
- Algorithm: Gradient Boosting Regressor
- R² Score: 0.726

## Tech Stack
- Python
- Scikit-Learn
- Pandas
- Streamlit
